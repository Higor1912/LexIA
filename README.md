# Chat Application

Este projeto é uma aplicação de chat que permite aos usuários interagir com um assistente jurídico. A aplicação é construída utilizando HTML, CSS e JavaScript, e se comunica com um servidor backend para processar as perguntas dos usuários.

## Estrutura do Projeto

```
chat-app
├── src
│   ├── assets
│   │   └── logo.png
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   ├── app.js
│   │   └── api.js
│   └── index.html
├── package.json
└── README.md
```

## Como Executar

1. **Clone o repositório:**
   ```bash
   git clone <URL_DO_REPOSITORIO>
   cd chat-app
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor backend:**
   Certifique-se de que o servidor backend está rodando na porta 3001.

4. **Abra o arquivo `index.html` em um navegador:**
   Você pode abrir o arquivo diretamente ou usar um servidor local para servir os arquivos.

## Funcionalidades

- Interface de chat interativa.
- Envio de mensagens para o assistente jurídico.
- Recebimento de respostas do servidor.
- Sugestões de perguntas pré-definidas.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Flet (para a interface do usuário)
