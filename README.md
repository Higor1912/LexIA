# Chat Application

Este projeto é uma aplicação de chat que permite aos usuários interagir com um assistente jurídico. A aplicação é construída utilizando HTML, CSS e JavaScript, e se comunica com um servidor backend para processar as perguntas dos usuários.


## Funcionalidades

- Interface de chat interativa.
- Envio de mensagens para o assistente jurídico.
- Recebimento de respostas do servidor.
- Sugestões de perguntas pré-definidas.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
- Flet (para a interface do usuário)
